`rstudio_bindings.json`: set keyboard shortcuts
- Reference: https://support.posit.co/hc/en-us/articles/206382178-Customizing-Keyboard-Shortcuts-in-the-RStudio-IDE
- File location
  - RStudio 1.2 and prior: `~/.R/rstudio/keybindings/rstudio/rstudio_bindings.json`
  - RStudio 1.3+, Windows: `~/AppData/Roaming/RStudio/keybindings/rstudio_bindings.json`
  - RStudio 1.3+, MacOS/Linux: `~/.config/rstudio/keybindings/rstudio_bindings.json`
- Can also be modified within RStudio: Tools > Modify Keyboard Shortcuts
